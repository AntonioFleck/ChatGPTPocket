Welcome to the Pocket GPT Chat Extension for Google Chrome. This extension provides an easy way to connect to OpenAI's GPT-3 API and create a pocket version of the GPT Chatbot. To get started, please refer to the documentation page, which explains how to generate a new API key for OpenAI. You can find the documentation here:

https://docs.openai.com/

Once you have generated your API key, you can then set up the Pocket GPT Chat Extension in your browser. This will allow you to access the GPT-3 API and create your own pocket version of the GPT Chatbot. We hope you enjoy using the Pocket GPT Chat Extension for Google Chrome.
